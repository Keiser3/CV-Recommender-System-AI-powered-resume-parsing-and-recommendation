# CV Recommender System

A full-stack application that allows users to securely upload CVs, extract and parse their content using AI, and provide embedding-based recommendations.

## Features

- **Secure CV Upload:** Upload PDF CVs with API key validation.  
- **AI-Powered Parsing:** Extract text from CVs and parse them using an offline LLM.  
- **Embedding-Based Search:** Generate embeddings from CV data for similarity search and recommendations.  
- **Full-Stack Architecture:**  
  - **Backend:** FastAPI  
  - **Frontend:** React + Vite  
  - **Database:** SQLite  

## Tech Stack

- **Frontend:** React, Vite, TailwindCSS  
- **Backend:** FastAPI, Python, httpx  
- **AI:** Local LLM for CV parsing and embeddings  
- **Database:** SQLite
